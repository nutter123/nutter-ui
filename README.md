## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 + less + element-ui

## 项目运行

``` 
git clone https://github.com/nutter123/nutter-ui

cd nutter-ui

npm install 或 yarn(推荐)

npm run dev ()

访问: http://localhost:8080

```
